# Cairn

Live meeting transcription with speaker diarization. Mac app + n4 service.

See spec: `~/fleet/docs/superpowers/specs/2026-05-07-cairn-design.md`

## Quickstart
- `npm install`
- `npm start` — launch Mac app (live capture mode)
- `npm run test:benchmark` — run end-to-end benchmark against pre-generated WAV
